# GA7-220501096-AA2-EV01 — Codificación de módulos con JDBC (Java)

Proyecto: Gestor de Tareas con JDBC (SQLite). Incluye CRUD (insertar, consultar, actualizar, eliminar),
estándares de codificación y estructura Maven.

**Repositorio remoto (Git):** https://github.com/MiguelTorres21/Programa-git

## Requisitos
- Java 17+
- Maven (o mvnd)

## Ejecutar
mvnd -q clean package
mvnd -q exec:java -Dexec.mainClass="com.migueltorres.tareas.Main"
